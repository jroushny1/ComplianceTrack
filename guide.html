1 What's Included

ComplianceTrack is a full applicant tracking system designed for solo compliance recruiting in financial services. Everything runs in your web browser with zero monthly cost.

👥

People Database

Store candidates with contact info, current employer/title, certifications (Series 7, 63, 66, SIE, etc.), licenses, status tracking, and notes.

💼

Jobs & Clients

Track open positions with required certs, salary range, location, and status. Manage client companies with contacts and billing info.

📊

Pipeline Kanban

Drag-and-drop candidates through hiring stages: Sourced, Contacted, Screening, Interview, Offer, Placed. Visual board per job.

✉

Outreach & Activity Log

Log every email, call, interview, note, and submission. Track response status (sent, replied, no response, bounced).

📧

Email Templates

Save reusable outreach templates with placeholders like {{firstName}}, {{currentEmployer}}, {{jobTitle}} that auto-fill per candidate.

⏰

Follow-Up Alerts

Set follow-up dates on activities. Dashboard shows overdue and upcoming follow-ups so nothing falls through the cracks.

💾

Import & Export

Full JSON backup/restore of all data. CSV import for bulk candidate loading. CSV export for spreadsheet use.

🌐

Works Offline

2 How to Install on Your PC

ComplianceTrack is a web app hosted online. You access it through your browser — there's nothing to download or install from GitHub. You can optionally "install" it as a desktop app.

Option A: Use It in the Browser (Easiest)

1

Open the app in Chrome or Edge

Go to the address below. Bookmark it for quick access.

https://jroushny1.github.io/ComplianceTrack/
2

That's it

The app loads in your browser. Once it loads the first time, it works offline too. Bookmark the page or pin the browser tab.

Option B: Install as a Desktop App (Recommended)

This puts a ComplianceTrack icon on your desktop/taskbar that opens in its own window — no browser toolbar or tabs.

1

Open the app in Chrome or Edge

Go to jroushny1.github.io/ComplianceTrack/

2

Install the app

In Chrome: Click the install icon (⬇) in the address bar, or go to the three-dot menu → "Install ComplianceTrack"

In Edge: Click the three-dot menu → Apps → "Install this site as an app"

3

Launch from your desktop

ComplianceTrack now appears as a regular app on your desktop, taskbar, and Start menu. It opens in its own window.

No GitHub account needed

You do not need a GitHub account, a repository clone, or any developer tools. The app is hosted online — you just visit the URL.

All data stays on your machine. Nothing is sent to any server.

Your data is stored inside your browser

ComplianceTrack uses a browser database called IndexedDB. This is a permanent storage area on your hard drive, managed by your browser (Chrome, Edge, etc.). Your candidates, jobs, clients, pipeline, and activity history all live here.

Your data is NOT stored online

Nothing goes to GitHub, the cloud, or any server. The GitHub Pages site only hosts the app code (HTML, CSS, JavaScript). Your actual recruiting data never leaves your computer.

Your data is tied to one browser on one computer

If you use Chrome on your desktop, that's where your data lives. It won't appear in Edge on the same computer, or in Chrome on a different computer. It's specific to that one browser profile.

What can delete your data

Clearing browser data ("Clear browsing data" with "Site data" or "Cookies and site data" checked) will erase your ComplianceTrack database. Regular history clearing is fine — just don't check the box for site data.

Uninstalling the browser or using a browser cleaning tool (like CCleaner) can also wipe IndexedDB.

Back up weekly

Because your data lives only in the browser, you should export a JSON backup regularly. This takes 5 seconds and saves everything.

4 Backup & Restore

Export a Backup (Do This Weekly)

1

Go to Settings

Click Settings in the sidebar (gear icon at the bottom).

2

Click "Export JSON Backup"

A file named compliancetrack-backup-YYYY-MM-DD.json downloads. This contains all your candidates, jobs, clients, pipeline entries, activities, and templates.

3

Save it somewhere safe

Move the file to a folder you won't accidentally delete — OneDrive, Google Drive, a USB drive, or a "Backups" folder on your desktop.

Restore from Backup

1

Go to Settings → "Restore JSON Backup"

Select your backup file. The app merges the backup data with anything already in the database (it won't create duplicates of records with the same ID).

Moving to a new computer?

Export a JSON backup from your old computer, open ComplianceTrack on the new computer, and restore the backup. All your data transfers over.

5 Importing Candidates

CSV Import (Bulk Loading)

If you have a spreadsheet of candidates (from Loxo, Excel, Google Sheets, etc.), export it as a CSV and import it into ComplianceTrack.

Required CSV columns

The CSV must include firstName and lastName columns. Optional columns: email, phone, currentTitle, currentEmployer, location, certifications, licenses, status, notes, externalId

1

Prepare your CSV

Open your spreadsheet and make sure the column headers match the names above. Save/export as CSV.

2

Go to Settings → "Import CSV (People)"

Select your file. The app previews what it found and imports the records.

Resume import is not supported

ComplianceTrack cannot parse PDF or Word resumes to extract candidate data. Parsing resumes requires AI/cloud services that would add ongoing costs. The workflow is: read the resume yourself, then add the candidate manually or via CSV.

6 Tips

Use Chrome or Edge

These browsers have the best support for PWA installation and IndexedDB. Firefox works for the app itself but doesn't support the "install as app" feature.

Don't use Incognito/Private mode

Private browsing windows don't persist IndexedDB data. Always use a regular browser window.

One browser = one database

If you start using ComplianceTrack in Chrome, stick with Chrome. Your data doesn't sync between browsers. If you need to switch, export a backup from the old browser and import it in the new one.

Set a weekly backup reminder

Put a recurring 5-minute calendar event on Friday afternoon: "Export ComplianceTrack backup." Save the JSON file to cloud storage (OneDrive, Google Drive). This protects you against browser data loss.

Use keyboard shortcuts

Press G then C to jump to People, N to create a new record, / to search. Much faster than clicking through menus.
